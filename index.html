Firebase 설정이 필요합니다

데이터베이스 연결을 사용할 수 없어 학생 데이터를 불러올 수 없습니다.

  1. Firebase 콘솔에서 프로젝트를 만들고 웹앱을 등록합니다.
  2. 좌측 메뉴 Build → Firestore Database 에서 데이터베이스를 생성합니다 (위치: asia-northeast3 권장).
  3. 받은 설정 객체를 프로젝트의 firebase-config.js 안의 YOUR_* 자리에 붙여넣고 저장합니다.
  4. 이 페이지를 새로고침합니다.

자세한 보안 규칙·운영 가이드는 README.md 의 "Firebase 연동 설정" 섹션을 참고하세요.
